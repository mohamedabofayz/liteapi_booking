# وحدة لايت إبي للحجز الفندقي - LiteAPI Booking لنظام أودو 17

**LiteAPI Booking Module - Foundation & Safety Layer for LiteAPI Integration**

## معلومات الوحدة
- **الاسم**: LiteAPI Booking
- **الإصدار**: 17.0.1.0.0
- **المؤلف**: Antigravity
- **الفئة**: Connector
- **الرخصة**: LGPL-3

## نظرة عامة

وحدة لايت إبي للحجز الفندقي هي وحدة متقدمة مصممة لدمج خدمات الحجز الفندقي من خلال واجهة برمجة تطبيقات (API) مع نظام أودو 17. توفر هذه الوحدة طبقة أساسية وآمنة لدمج خدمات الحجز الفندقي من لايت إبي، وتشمل ميزات البحث عن الفنادق وإدارة الحجوزات وقدرات التقارير.

## الميزات الرئيسية

### 1. إدارة الحجوزات الفندقية
- تسجيل الحجوزات الفندقية مع تفاصيل كاملة
- تتبع حالة الحجز (معلق، مؤكد، فاشل، ملغى)
- ربط الحجز بطلب المبيعات وبيانات العميل

### 2. نموذج الحجز
- **الحقل الأساسي**: `name` - مرجع الحجز (يتم إنشاؤه تلقائيًا)
- **العلاقات**:
  - `partner_id`: عميل الحجز
  - `sale_order_id`: طلب المبيعات المرتبط
  - `hotel_id`: الفندق المحجوز

### 3. تفاصيل الحجز
- معرف الحجز من لايت إبي (`liteapi_booking_id`)
- معرف الحجز المؤقت (`prebook_id`)
- معرف المعاملة (`transaction_id`)
- اسم الفندق ونوع الغرفة
- تواريخ الوصول والمغادرة
- عدد الضيوف
- اسم الضيف والبريد الإلكتروني
- السعر الإجمالي والعملة

### 4. سياسة الإلغاء والاسترجاع
- دعم الحجوزات القابلة للاسترداد وغير القابلة للاسترداد
- تحديد موعد نهائي للإلغاء
- سياسة إلغاء مفصلة

### 5. واجهة المستخدم
- واجهات بصرية لإدارة الحجوزات
- قوالب بوابة للعملاء
- قوائم وأوامر مخصصة
- أدوات الإبلاغ والإحصاءات

### 6. التكامل
- وظائف تحكم لواجهة الويب
- دعم عمليات الدفع والCheckout
- وظائف بحث متقدمة للفنادق
- إدارة المدن والدول

### 7. الأمان والجودة
- تدقيق API لمراقبة الاستخدام
- تقارير إساءة الاستخدام
- لوحة تحكم إدارية
- تكامل مع Cron للوظائف المجدولة

## التركيب
هذه الوحدة متوافقة مع أودو 17 ويمكن تركيبها كتطبيق. تأكد من أن جميع التبعيات مثبتة قبل تركيب الوحدة.

## الاستخدام
1. بعد تركيب الوحدة، يمكن للمستخدمين بدء البحث عن الفنادق من خلال واجهة البوابة
2. يتم إنشاء الحجوزات تلقائيًا مع تفاصيل كاملة
3. يمكن تتبع الحجوزات من خلال لوحة الإدارة
4. تتوفر خيارات الإلغاء والاسترجاع حسب سياسة الفندق
